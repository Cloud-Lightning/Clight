# HPM Boards

Generated HPM board packages belong here in local projects:

```text
platforms/hpm/boards/<board_name>/
```

Typical files:

```text
board.c
board.h
clock.c
clock.h
pinmux.c
pinmux.h
CMakeLists.txt
<board_name>.yaml
<board_name>.cfg
```

Do not commit `.hpmpc` files or local credentials.
