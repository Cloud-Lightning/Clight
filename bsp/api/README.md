# Clight C++ API

Public C++ API headers are generated by `tools/clight_codegen/generate_cpp_api.py`
directly into this directory.

Application and module code should include headers from this unified include path,
for example `Gpio.hpp`, `Spi.hpp`, `Uart.hpp`, `Can.hpp`, and `BoardIds.hpp`.
Do not include platform-specific paths such as `bsp/api/stm32` or `bsp/api/hpm`.
